## Problem Statement

Users abandon onboarding at step 3 because they don't understand the value proposition before being asked to connect integrations.

## Who Has It

**Segment:** New signups in first 7 days
**Context:** Particularly those without prior SaaS tool experience

## What They've Tried

Reading the help docs, emailing support.

## Competitive Landscape

| Competitor | Approach | Differentiation opportunity |
|---|---|---|
| Competitor A | Skips integrations step entirely, shows value first | Lead with value before asking for integration |
| Competitor B | Uses video walkthrough to explain value | Interactive demo instead of passive video |

## Hypotheses

- [ ] If users see a concrete outcome (first report generated) before connecting integrations, they will complete onboarding at a higher rate
- [ ] Reducing required steps from 5 to 3 in the first session will increase 7-day retention

## UX Research

**User journeys:** User signs up → sees blank dashboard → is prompted to connect integrations → abandons

**Mental models:** Users expect to see value before committing to setup effort

**Failure points:** Step 3 integration prompt — 58% drop-off rate at this step

## Success Metrics

| Metric | Baseline | Target | Threshold (promote) | Threshold (kill) | Window |
|---|---|---|---|---|---|
| onboarding_completion_rate | 0.42 | 0.60 | 0.55 | 0.44 | 14 days |

## Value Estimate

**Effort:** M
**Opportunity:** +18% trial-to-paid conversion if onboarding completion reaches target
**Confidence:** high

## Reviewer Verdict
